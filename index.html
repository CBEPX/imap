Imap

Interactive map for Zabbix

Download this project as a .zip file Download this project as a tar.gz file

Важно: Аддон не вносит никаких изменений в существующие базу и данные. В любой момент вы удаляете файлы, папку и внесенные изменения и все возвращается на круги своя.

Сам аддон содержит папку imap и файл imap.php, а также файлы добавочной локализации (на существующую локализацию не влияет).

Распакуйте их в папку с web-мордой zabbix (типа, /usr/share/zabbix). Выставите правильные права (на чтение для web-сервера). Если сразу открыть imap.php в браузере, вы получите сообщение, типа:

Доступ запрещен. Вы подключены как admin. У вас нет полномочий для доступа к этой странице.

Нужно добавить ссылку на карту в интерфейс

Чтобы встроить ссылку на карту в интерфейс, придется немного поправить один системный файл.

Нас интересует файл include/menu.inc.php Пример встраивания ниже.

array(

'url' => 'srv_status.php',

'label' => _('IT services'),

'force_disable_all_nodes' => true,

'sub_pages' => array('report3.php', 'chart5.php') ),

array(

'url'=>'imap.php',

'label'=>_('Interactive map')

),

array(

'url' => 'chart3.php'
